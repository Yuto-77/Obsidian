## 📝 概要
オブジェクトの「設計図」または「ひな形」

## 📚 詳細解説
- データ（[[フィールド]]や[[プロパティ]]）と、そのデータを操作する機能（[[メソッド]]）をひとまとまりにして定義します。
- プログラムの構造を整理し、コードを再利用しやすくします。
- 複雑なシステムを小さな部品（[[オブジェクト]]）に分割して考えることができるようになります。

## 💻 C#での使用例

```csharp
class クラス名
{
	クラスの実装
}
```
